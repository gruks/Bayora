    1. Implement HHRLHFClassifier:
       - Create `services/blue-agent/src/blue_agent/classifiers/hh_rlhf_classifier.py`
       - Class `HHRLHFClassifier(SafetyClassifier)` that loads a DistilBERT sequence classification model
       - __init__ accepts `model_path: str` (path to fine-tuned model directory) and optional `device: str = "cpu"`
       - Use `transformers.pipeline("text-classification", model=model_path, tokenizer=model_path, device=device)` for inference
       - `classify(self, response_text: str) -> ClassificationResult`:
         - Run pipeline on response_text
         - Map pipeline output (label "LABEL_0"/"LABEL_1") to "harm"/"safe"
         - Return ClassificationResult with label, confidence (from pipeline score), model_source="hh_rlhf"
       - Handle model loading errors gracefully (raise ValueError with clear message if model_path invalid)

    2. Create training script:
       - Create `services/blue-agent/scripts/train_hh_rlhf.py`
       - Script uses `datasets.load_dataset("Anthropic/hh-rlhf", data_dir="harmless-base")`
       - Convert preference pairs to binary labels: chosen -> label=1 (safe), rejected -> label=0 (harm)
       - Tokenize with `distilbert/distilbert-base-uncased` tokenizer (truncation=True, max_length=512)
       - Fine-tune using `transformers.Trainer` with TrainingArguments:
         - learning_rate=2e-5, per_device_train_batch_size=16, num_train_epochs=3, weight_decay=0.01
         - eval_strategy="epoch", save_strategy="epoch", load_best_model_at_end=True
         - output_dir from CLI argument (default: "./models/hh-rlhf-classifier")
       - Add CLI argument `--output-dir` for model save path
       - Add note in docstring: "Requires GPU for reasonable training times. Use Google Colab if local GPU unavailable."
       - Add entry point in pyproject.toml: `train-hh-rlhf = "scripts.train_hh_rlhf:main"`

    3. Write tests in `services/blue-agent/tests/test_classifiers.py`:

       - Test that classify() accepts only response_text (no prompt parameter)
