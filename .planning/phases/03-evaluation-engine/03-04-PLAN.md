<task type="auto">
  <name>Task 1: Implement BCa bootstrap statistics and multi-seed evaluator</name>
  <files>
    services/blue-agent/src/blue_agent/evaluation/__init__.py
    services/blue-agent/src/blue_agent/evaluation/runner.py
    services/blue-agent/src/blue_agent/evaluation/statistics.py
    services/blue-agent/pyproject.toml
  </files>
  <action>
    1. Add scipy dependency to blue-agent/pyproject.toml:
       - `scipy>=1.14`

    2. Implement BCa bootstrap statistics:
       - Create `services/blue-agent/src/blue_agent/evaluation/statistics.py`
       - Define `compute_confidence_intervals(per_seed_scores: list[float], confidence_level: float = 0.95, n_resamples: int = 10000) -> tuple[float, float]`
       - Uses `scipy.stats.bootstrap` with method="BCa"
       - Returns (ci_low, ci_high)
       - Handle edge case: fewer than 3 seeds raises ValueError with message "Minimum 3 seeds required for BCa bootstrap"
       - Define `compute_statistics(per_seed_scores: list[float]) -> dict` returning {mean, std, median, min, max, ci_low, ci_high, n_seeds}

    3. Implement MultiSeedEvaluator:
       - Create `services/blue-agent/src/blue_agent/evaluation/runner.py`
       - Define `MultiSeedEvaluator` class:
         - __init__ accepts `evaluator: BlueAgentEvaluator`, `metric_registry: MetricRegistry`, `default_seeds: list[int] | None = None`
         - If default_seeds is None, use [42, 123, 456, 789, 1011] (5 seeds minimum)
         - Store both evaluator (for classification) and registry (for direct metric scoring)
       - `evaluate(self, responses: list[str], seeds: list[int] | None = None) -> dict`:
         - For each seed: set np.random.seed(seed), run evaluator.evaluate_response() on each response
         - Also call metric_registry.score(response_text) directly for each response (this is the key_link to registry)
         - Collect per-seed scores (mean safety score, pass rate, classification results)
         - Compute BCa bootstrap CIs for each metric
         - Compute classifier-dependent derived metrics from EvaluationResult data:
           - `harm_classification_confidence`: extract from evaluation_result.classification.confidence when label="harm"
           - `toxigen_confidence`: extract from sub-classifier results if ensemble classifier was used
           - `ensemble_agreement`: compare HH-RLHF and ToxiGen labels from sub_results (if available)
         - Return dict with structure:
           ```
           {
             "n_responses": int,
             "n_seeds": int,
             "seeds": list[int],
             "overall": {mean, std, ci_low, ci_high, by_category},
             "per_seed": [{"seed": int, "mean": float, "pass_rate": float, ...}],
             "per_response": [{"response_index": int, "classification": ..., "metrics": ..., "ci": ...}],
             "classifier_derived": {
               "harm_classification_confidence": float,
               "toxigen_confidence": float | None,
               "ensemble_agreement": float,
             },
           }
           ```
       - `evaluate_single(self, response_text: str, seeds: list[int] | None = None) -> dict`:
         - Convenience method for single-response evaluation

    4. Create evaluation/__init__.py exporting MultiSeedEvaluator
  </action>
