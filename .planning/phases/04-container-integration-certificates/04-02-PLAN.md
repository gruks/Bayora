<task type="auto">
  <name>Document gVisor Windows fallback</name>
  <files>README.md</files>
  <action>
    Add to project README or create docs/gvisor-setup.md:

    **gVisor Installation:**
    ```bash
    # Linux
    curl -O https://gvisor.dev/releases/runsc
    chmod +x runsc
    sudo mv runsc /usr/local/bin/
    runsc install
    ```

    **Docker daemon.json:**
    ```json
    {
      "runtimes": {
        "runsc": {
          "path": "/usr/local/bin/runsc",
          "runtimeArgs": ["--net-raw", "--allow-packet-socket-write", "--platform=systrap"]
        }
      }
    }
    ```

    **Windows/macOS Fallback:**
    - Use Docker Desktop with Linux containers (WSL2 backend on Windows)
    - Or deploy with native Docker runtime (reduced isolation):
      ```yaml
      llm-sandbox:
        runtime: runc  # Instead of runsc
      ```
    - Note: ARCH-06 (gVisor) requires Linux; native runtime provides basic isolation only

    This documents the blocker from STATE.md.
  </action>
  <verify>
    grep -r "gVisor\|runsc\|Windows" README.md 2>/dev/null | head -5
  </verify>
  <done>
