    Create configs/seccomp/ directory and add JSON seccomp profiles for each service:

    **red-agent.json:** Allow system calls needed for LLM API calls, HTTP requests, logging. Block: mount, kexec, init_module.

    **orchestrator.json:** Allow system calls for HTTP server, process spawning, database, logging. Block: mount, kexec, init_module.

    Apply via security_opt in docker-compose.yml: seccomp:/configs/seccomp/{service}.json
  </action>
